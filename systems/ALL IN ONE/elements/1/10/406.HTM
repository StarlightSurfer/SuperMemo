一、Redis 数据库操作

1. 数据模型设计

  • 键命名规范：采用 实体类型:ID 的层级结构（如 Student:S01、Course:C01）。
  • 数据类型选择：
    • 使用 String 存储 JSON 对象（如学生信息、课程信息）。
    • 复杂关系可考虑 Hash 或 Set（本题使用 String 简化操作）。

2. 核心命令

命令 用途 示例
SET key value 设置键值对 SET Student:S01 '{"Sno":"S01","Sname":"zhangli"}'
GET key 获取键对应的值 GET Student:S01
KEYS pattern 查找匹配的键 KEYS Report:S02:* （查找学号 S02 的所有成绩记录）
--raw 选项 以原始格式显示结果 redis-cli --raw GET Student:S01 （正确显示中文）

3. 实际操作技巧

  • 中文显示问题：
    • 使用 --raw 选项避免 Unicode 转义（如 \xe5\xa5\xb3）。
    • 结合 jq 工具解析 JSON：redis-cli GET Student:S01 | jq '.Sage'。
  • 数据验证：
    • KEYS * 查看所有键。
    • TYPE key 检查键的数据类型。

二、MongoDB 数据库操作

1. 数据模型设计

  • 集合设计：
    • students：存储学生信息（字段：Sno, Sname, Sage, College）。
    • courses：存储课程信息（字段：Cno, Cname, Cterm, Credits）。
    • reports：存储成绩信息（字段：Sno, Cno, Grade）。
  • 文档结构：采用 BSON 格式（类 JSON），支持嵌套。

2. 核心命令

命令 / 操作 用途 示例
数据插入
db.collection.insertOne() 插入单个文档 db.students.insertOne({Sno:"S01", Sname:"zhangli"})
db.collection.insertMany() 批量插入文档 db.courses.insertMany([{Cno:"C01"}, {Cno:"C02"}])
数据查询
db.collection.find() 查询文档 db.students.find({Sage: {$gt: 20}}) （查询年龄 > 20 的学生）
db.collection.findOne() 查询单个文档 db.students.findOne({Sno:"S02"})
聚合操作
$group 分组统计 db.reports.aggregate([{$group: {_id: "$Cno", avgGrade: {$avg: "$Grade"}}}])
索引管理
createIndex() 创建索引 db.students.createIndex({Sname: 1}, {unique: true}) （唯一索引）
正则表达式 模糊查询 db.students.find({Sname: /^李/}) （查询姓李的学生）

3. 实际操作技巧

  • 查询优化：
    • 使用投影参数限制返回字段：db.students.find({}, {Sname: 1, _id: 0})。
    • 复合条件查询：db.students.find({Sage: {$gt: 20}, College: "信息学院"})。
  • 索引应用：
    • 对高频查询字段（如 Sname、Sno）创建索引提升性能。
    • 唯一索引可防止数据重复（如 Sno 字段）。

三、对比总结

特性 Redis MongoDB
数据模型 键值对（支持多种数据类型） 文档型（BSON 格式）
查询能力 简单查询（通过键） 复杂查询（支持聚合、索引）
适用场景 缓存、会话存储、排行榜 结构化数据存储、复杂查询
数据持久化 支持 RDB/AOF 快照 支持 WiredTiger 存储引擎
分布式支持 Redis Cluster MongoDB Sharding

四、常见问题处理

  1. Redis 中文显示异常：
    • 使用 --raw 选项或 jq 工具解析。
  2. MongoDB 权限配置：
    • 生产环境启用认证：security.authorization: enabled。
  3. 数据导入导出：
    • Redis：SAVE 生成 RDB 快照，RESTORE 恢复。
    • MongoDB：mongodump/mongorestore 或 mongoexport/mongoimport。
